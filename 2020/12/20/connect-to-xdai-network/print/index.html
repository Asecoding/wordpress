Connect to xDai Network

Custom RPC Settings

Open Metamask, and select “Custom RPC” from the Network Dropdown.

In the “Custom RPC” Settings, add in the xDai network details and click Save:

Network Name: xDai
New RPC URL: https://rpc.xdaichain.com/[1]
Chain ID: 0x64 (100)
Symbol: xDai
Block Explorer URL: https://blockscout.com/poa/xdai[2]

Endnotes:
  1. https://rpc.xdaichain.com/: https://rpc.xdaichain.com/
  2. https://blockscout.com/poa/xdai: https://blockscout.com/poa/xdai
