Adding TriniSwap Token

Adding TriniSwap Token to Metamask

Open Metamask, and select “Add Token” from the bottom of the Metamask window.

Click “Custom” tab.

Type in the following information to add TriniSwap Token to your list

Token Contract Address: 0x3e9aac491d04bb8930b4ca558fa027961e7ee416
Symbol: TRINI
Decimals: 18
